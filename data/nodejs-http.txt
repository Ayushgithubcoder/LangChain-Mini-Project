HTTP Server in Node.js

Node.js can create HTTP servers using the built-in http module.

Basic HTTP Server:
const http = require('http');

const server = http.createServer((req, res) => {
  res.writeHead(200, { 'Content-Type': 'text/plain' });
  res.end('Hello World');
});

server.listen(3000, () => {
  console.log('Server running on port 3000');
});

You can also use frameworks like Express.js for more advanced routing and middleware.
